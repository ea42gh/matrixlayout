JupyterLab Notebooks
====================

Use this directory for notebooks that demonstrate `matrixlayout` features.

Guidelines
----------
- Keep notebooks short and focused.
- Include a brief header cell describing the goal.
- Avoid large binary outputs in version control.

Notebooks are referenced from MkDocs via `mkdocs-jupyter`.

Notebook outputs should be cleared before committing unless the output is
essential for documentation.

## Notebook Index

- 01_ge_grids.ipynb: GE grid APIs (render_ge_tex/render_ge_svg/bundle/specs).
- 02_qr_grids.ipynb: QR grid APIs.
- 03_eig_svd_tables.ipynb: Eigen/SVD table renderers.
- 04_backsubstitution.ipynb: Backsubstitution renderers.
- 05_formatting_selectors.ipynb: Formatting + selectors.
- 06_rendering_templates.ipynb: Rendering + Jinja helpers.
