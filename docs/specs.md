# Specs

This page describes the layout specs consumed by matrixlayout.

## GE specs

- `matrices`: grid of matrices (list of rows)
- `Nrhs`: number of RHS columns for augmented matrices
- `preamble`, `extension`, `nice_options`
- `pivot_locs`, `rowechelon_paths`, `callouts`

## QR specs

- `matrices`
- `array_names`
- `decorators`

## Eigen/SVD specs

- `lambda`, `ma`
- `evecs`, `qvecs`, `uvecs`
- `sigma`, `sz`
