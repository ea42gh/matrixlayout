# Glossary

- **Block**: A single matrix placed in a grid cell.
- **Grid**: A 2D list of blocks representing a layout.
- **Spec**: A dictionary describing layout options and annotations.
- **Callout**: A label attached to a named submatrix delimiter.
- **Decorator**: A TeX transform applied to formatted entries.
- **Selector**: A specification of which entries a decorator targets.
- **Pivot box**: A rectangular highlight around a pivot entry.
